# Conversor de Texto

Converte seu texto escrito em maiúsculo, minúsculo, correção ortográfica do navegador, com a primera letra em maiúsculo e texto invertido. 
Uso no meu dia-a-dia. Nenhuma informação é armazenada, apenas serve para alterações no texto para a área de transferência.
